## TODOWIZARD
 It is a simple Todo-list Application
## DESCRIPTION
  ToDoWizard is a basic to-do list app that helps you keep track of your tasks.
## FEATURES
 - Add tasks
 - Delete tasks
 - Tasks are saved locally
 - Tasks stay on the page until deleted
## USING TECHNOLOGIES
 1. HTML
 2. CSS
 3. JAVASCRIPT
